| Operator | Name                  | Description                 | Example     | Result  |
| -------- | --------------------- | --------------------------- | ----------- | ------- |
| `==`     | Equal to              | Compares **values only**    | `5 == "5"`  | `true`  |
| `===`    | Strict equal          | Compares **value and type** | `5 === "5"` | `false` |
| `!=`     | Not equal             | Values are not equal        | `5 != 3`    | `true`  |
| `!==`    | Strict not equal      | Value or type not equal     | `5 !== "5"` | `true`  |
| `>`      | Greater than          | Left value is greater       | `10 > 5`    | `true`  |
| `<`      | Less than             | Left value is smaller       | `5 < 10`    | `true`  |
| `>=`     | Greater than or equal | Greater or equal            | `10 >= 10`  | `true`  |
| `<=`     | Less than or equal    | Less or equal               | `5 <= 10`   | `true`  |
